### Setup

#### Running the app
* config .env variables:
    * PORT
    * MONGODB_URI
    * JWT_SECRET

* start a MongoDB instance with the data from your local mongo db folder: `mongod --dbpath=./data`
* `node index.js`
* http://localhost:3000

  
#### Tests
* Unit Tests: 'npm test'
* Lint Tests: 'npm run lint'
